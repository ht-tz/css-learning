• flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]
  • 1.flex 属性是 flex-grow、flex-shrink 和 flex-basis 缩写，默认值是 0 1 auto，即不增长，但收缩，收缩比例为 1，flex-basis 为 auto，即取自用户定义的宽度或内容的宽度。
  • 2.指定一个数字 - 例如flex: 1，就等同于是flex: 1 1 0，即自动缩放，比例为 1，flex-basis 为 0。 auto - 等同于flex: 1 1 auto。
  • 3.指定两个数字 - 第一个为flex-grow，第二个，如果是数字则认为是 flex-shrink，如果是宽度，则是flex-basis。
  • 4.指定三个值 - 分别为flex-grow，flex-shrink 和 flex-basis。
